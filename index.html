📊 ตั้งค่า Google Sheets
วางลิ้ง Apps Script Web App URL ด้านล่าง เพื่อ sync ข้อมูลอัตโนมัติ
📋 วิธีสร้าง Apps Script (คลิกเพื่อดู)
ขั้นตอน:
1. เปิด sheets.google.com → สร้าง Spreadsheet ใหม่
2. คัดลอก URL ของ Sheet มาวางที่ช่อง "Google Sheet URL" ด้านบน
3. ใน Sheet → ไปที่ Extensions → Apps Script
4. ลบโค้ดเดิม → วางโค้ดด้านล่างนี้ทั้งหมด
5. กด Deploy → New deployment
6. Type: Web app | Execute as: Me | Access: Anyone
7. Copy URL มาวางที่ช่อง "Apps Script Web App URL"
